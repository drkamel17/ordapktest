Accident aux AVK

Clinique

  • Syndrome hemorragique

Bilan :

  • TP
  • INR
  • Rechercher une Insuffisance Hepatique / Renale
  • Arret de Sintrom
  • Traitement par Vit K 1amp (5mg) en per Os

Si hemorragies grave + INR>10

  • Vit K 10mg
  • Perfusion de plasma frais congele

Surveillance

  • Surveillance de TP et INR
  • Si dans l'intervalle => Avis Hematologie
  • Chaque Pathologie => INR cible
INR augmente : Risque hemorragique => Diminuer AVK
INR diminue : Risque de Thrombose => Augmenter AVK

Conduite selon INR

  • INR (2-3) : Surdosage asymptomatique
  • INR (3-4) : Ajustement de posologie
  • INR (4-6) : On saute une prise d'AVK
  • INR (6-10) : Arret du traitement => 1-2mg de Vit K en per os
  • >10 : Arret du traitement => 5mg de Vit K
Toujours controle INR le lendemain
