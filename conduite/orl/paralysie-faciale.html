Paralysie Faciale Périphérique

Signes cliniques

  • Déviation buccale vers le côté sain
  • Effacement du sillon naso-génien
  • Impossibilité de souffler et gonfler les joues
  • Effacement des rides frontales
  • Impossibilité de fermer l'œil +++
  • Impossibilité de relever les sourcils
  • Signe de Charles Bell : fermeture incomplète de l'œil

Différence PF centrale vs périphérique

  • Centrale : atteinte du territoire inf → Neuro
  • Périphérique : atteinte de l'hémiface → on traite

Étiologies

  • ORL : traumatique, post otite, parotidite
  • Idiopathique : inflammatoire, ischémique, infectieuse

Traitement

  • 1) Inflammatoire : Solupred 1mg/kg/j, 05 jours à dose dégressive
  • 2) Ischémique : Aspegic sachet 100mg/j + Praxilene cp 200 x 2/j
  • 3) Infectieuse : Aciclovir cp 200 : 01 cp x 3/j
  • Neurovit cp : 01 cp x 2/j
  • Larmes artificielles : 01 gtte x 4/j
  • Pansement oculaire
  • Lettre pour rééducation fonctionnelle après 48h

CI

  • Chewing-gum
