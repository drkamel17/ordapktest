Hyperglycemie (sans cetose / avec cetose)

Clinique

  • Sujet diabetique: mauvaise prise du traitement ou infection
  • Sujet non diabetique: Vertige, asthenie

CAT - Adulte

  • Dextro > 2.5 g/l : Labstix corps cetoniques (-)
  • Ionogramme avant Insulinotherapie pour corriger une Hypokaliemie (Traitement preventif durant ces 24h)
  • Insulinotherapie => 10 Ui en Sc/h ou 0.1 Ui/kg/h
  • Si corps cetoniques (+) => 10 Ui en IV/h jusqu'a (-) des CC
  • Des (+) => 5 Ui en IVD et 5 Ui en Sc (hiatus)
  • Puis apres 6h : 10 Ui/6h en Sc
  • Ou 0.1 Ui/kg si (-) des CC => 0.05 Ui/kg jusqu'a 24h

Rehydratation

  • Boire 1.5l eau dans 2h
  • Perfusion SSI : 50 cc/kg/24h
  • 1l pdt 1h puis 1l pdt 2h puis 1l pdt 3h puis 3l pdt 6h
  • 1amp de K dans le flacon a partir de la 4eme heure
  • Gly < 2.5g/l => SG5%
  • Gly < 1.5g/l => SG10%
  • Gly < 1g/l => SG30%

Indications hospitalisation

  • Si facteurs de risque + decouverte de Glycemie elevee
  • Si corps cetoniques (+) + Glycemie elevee

Apres 1h

  • Si CC(-) + Gly < 2g/l => Liberer avec traitement etiologique si infection
  • Si CC(-) + Gly > 2g/l => Continue meme schema d'insulinotherapie
  • Si CC(+) + Gly > 2g/l => Continue meme schema. Si non insuline a la seringue electrique
Ne jamais liberer un patient sans cessation des corps cetoniques

Examens

  • TLT : Rechercher un foyer infectieux
  • Labstix / ECBU
  • ECG : si Hypokaliemie corriger avant Insulinotherapie

Surveillance

  • Glycemie / 30 min et Labstix / 3h
  • Ionogramme / 3h

Bilan

  • Uree / Creatinine
  • Ionogramme
  • Si Gly > 3g/dl => 1/5 de la dose totale journaliere d'insuline
  • Si 2.5 < Gly < 3 => 1/10 de la dose totale journaliere

dose totale journaliere

  • Adulte : 1.5 Ui/Kg
  • Enfant : 1 Ui/Kg
  • NRS : 0.5 - 0.8 Ui/Kg
