Dyspnee

Clinique

  • Difficulte a respirer avec une sensation de gene ou oppression
  • Si cardiopathie: polypnee avec sensation de soif d'air
Valeurs normales de FR:
  • NN => 0_28j : 40_60/min
  • NRS => 28j_2ans : 30_60/min
  • Enfant < age de puberte : 22_30/min
  • Adulte : 14_22/min

Etiologies

  • OAP:
    • Detresse respiratoire brutale, nocturne
    • Polypnee superficielle rapide,angoisse +/_ douleur, agitation
    • Creatilents bilateraux
    • Gravite: hypoTA, cyanose
    • Dyspnee, Toux ramenent une expectoration mousseuse blanchatre..
  • Embolie pulmonaire:
    • Polypnee brutale avec douleur basithoracique violente
    • +/_ cyanose, syncope..
    • TVP, alitement prolonge, post op...
    • Examen clinique cardio pulmonaire normal
  • BPCO: Fumeur, toux avec expectorations, dyspnee pas amelioree a la position assise
  • Asthme: Terrain atopique, bradypnee expiratoire, sibilants, nocturne
  • Pneumopathies: Syndrome infectieux, fievre, syndrome de condensation parenchymateuse
  • Chez un sujet Obese:
    • Rechercher un diabete: Signes de resistance a l'insuline
    • Syndrome d'apnee de sommeil

Bilan: 🩸

  • TSH
  • Gly a jeun / HbA1C
  • Lipidique
  • Hepatique + Echographie hepatique => Rechercher une steatose hepatique
