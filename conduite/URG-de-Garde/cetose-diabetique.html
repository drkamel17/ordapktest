Cetose Diabetique

Definition

Acidocetose diabetique : complication metabolique grave du diabete insulinodependant caracterisee par une hyperglycemie, une cetonemie et une cetonurie.

Diagnostic

  • Glycemie : > 2,5 g/L
  • Acetoner : presente (+)
  • pH : < 7,35 (acidose metabolique)
  • Troubles ioniques : hyponatremie, hypokaliemie

Prise en Charge

Selon la glycemic et l'acetonurie

Glycemie > 2,5 g/L :

  • SSI a 9% + electrolytes

Glycemie < 2,5 g/L :

  • SGI a 5% + electrolytes

Insulinotherapie

Insuline (Actrapid)

  • Acetonurie (+) : 10 UI IV/1h
  • Acetonurie (-) : 10 UI SC/4h

Electrolytes

  • NaCl : 2g (1 ampoule)
  • KCl : 1g (1/2 ampoule)
  • Mg+2 : 1g (1/2 ampoule)
  • Ca+2 : 1g (1/2 ampoule)

Si signes d'infection

Antibiotherapie :

  • Metronidazole (Flagyl) 500mg/1 flacon /8h
  • Autres antibiotiques selon type d'infection (ex: Ceftriaxone Claforan /6h)

Surveillance

  • Glycemie capillaire : toutes les heures
  • Ionogramme : toutes les 4 a 6 heures
  • Diurese : toutes les heures
  • ETCO2 : surveillance continue
