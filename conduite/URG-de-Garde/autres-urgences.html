Autres Urgences

Convulsion

Si 1ere convulsion :

  • Recherche hypocalcemie, AVC, troubles ioniques
  • ECG
  • TDM cerebral

Si epileptique connu :

  • Recherche ecart therapeutique

Si pas de crise :

  • TDM cerebral + bilan urgence + surveillance

Si crise en cours :

  • Diazepam (Valium) 1 ampoule IVL
  • Puis Phenobarbital (Gardenal) 10 mg/kg IV

Ingestion de produit caustique

Bilan initial :

  • Telethorax + ASP

Si acide :

  • Arret alimentation
  • Rehydration
  • Ranitidine (Azantac)
  • Fibroscopie OGD precoce

Si eau de javel :

  • Arret alimentation
  • Rehydration
  • Ranitidine (Azantac)
  • Fibroscopie apres 48h

Traitement preventif :

  • IPP (Omeprazole) tritherapie ulcere

Pic Hypertensif

Definition : PAS > 210 mmHg ou PAD > 120 mmHg

Regle : Ne pas abaisser TA plus de 25% en 24h

Traitement :

  • Nicardipine (Loxen) max 10 mg/h
  • Surveillance continue

Douleur thoracique

Bilan :

  • TA, Glycemie, Telethorax, ECG
  • Avis cardiologue

Si ECG normal :

  • Mg2+ 1/2 ampoule IV

Si ECG pathologique :

  • Troponine, CPK

Hemorragie digestive

Bilan urgence (surtout HK) :

  • NFS, ionogramme, hemostase

Si HK < 25% :

  • Transfusion

Presentation :

  • Adresser a gastro pour fibro + hospitalisation

Si saignement actif :

  • Lavage SSI glace

Si TA basse :

  • Plasmagel
